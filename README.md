# Protein Sequence Analysis

A basic bioinformatics project analyzing breast cancer-related protein sequences from UniProt.

## Project Overview
- **Data Source**: UniProt (Breast cancer-related human proteins)
- **Analysis**: Sequence length distribution, amino acid composition, basic visualizations
- **Tools**: Python, Pandas, Biopython, Matplotlib

## Files
- `protein_analysis.py` - Main analysis script
- `requirements.txt` - Python dependencies
- `protein_analysis_results.png` - Output visualizations

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run analysis: `python protein_analysis.py`
